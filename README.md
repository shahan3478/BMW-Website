BMW Showroom Landing Page
A premium, interactive web application designed to showcase BMW's precision engineering and aesthetic appeal. This project provides a sleek, modern user experience where visitors can explore vehicle details and interact with dynamic visual elements.

🚀 Key Features
Dynamic Color Switching: Allows users to interactively toggle between different vehicle finishes—including Black, Blue, and Red—to visualize the car in real-time.  

Immersive UI: Features a high-end design with transparent image assets for a professional, "floating" visual effect.  

Fully Responsive: Built to ensure a seamless experience across desktops, tablets, and mobile devices.  

Vanilla Implementation: Developed using pure HTML, CSS, and JavaScript for optimal performance and minimal loading times.  

🛠️ Project Structure
The codebase is organized within the BMW-Website-main.zip archive and includes the following assets:  

index.html: The structural backbone of the showroom interface.  

style.css: Contains custom styling, layout positioning, and interactive transitions.  

script.js: Manages the logic for dynamic image switching and event handling.  

img/: A dedicated directory housing branding and high-fidelity vehicle assets:

logo.png: Official BMW branding.  

black-removebg-preview.png: Black color variant.  

blue-removebg-preview.png: Blue color variant.  

red-removebg-preview.png: Red color variant.  

💻 Tech Stack
HTML5: Semantic structure.

CSS3: Custom layouts and visual effects.

JavaScript (ES6+): Dynamic DOM manipulation and interaction.
