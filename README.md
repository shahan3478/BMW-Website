# BMW Showroom Landing Page

A premium, interactive web application designed to showcase BMW's precision engineering and aesthetic appeal. This project provides a sleek, modern user experience where visitors can explore vehicle details and interact with dynamic visual elements.

---

## 🚀 Key Features

*   **Dynamic Color Switching**: Allows users to interactively toggle between different vehicle finishes—including Black, Blue, and Red—to visualize the car in real-time[cite: 4].
*   **Immersive UI**: Features a high-end design with transparent image assets for a professional, "floating" visual effect[cite: 4].
*   **Fully Responsive**: Built to ensure a seamless experience across desktops, tablets, and mobile devices[cite: 4].
*   **Vanilla Implementation**: Developed using pure HTML, CSS, and JavaScript for optimal performance and minimal loading times[cite: 4].

---

## 🛠️ Project Structure

The codebase is organized within the **BMW-Website-main.zip** archive and includes the following assets[cite: 4]:

*   **`index.html`**: The structural backbone of the showroom interface[cite: 4].
*   **`style.css`**: Contains custom styling, layout positioning, and interactive transitions[cite: 4].
*   **`script.js`**: Manages the logic for dynamic image switching and event handling[cite: 4].
*   **`img/`**: A dedicated directory housing branding and high-fidelity vehicle assets:
    *   `logo.png`: Official BMW branding[cite: 4].
    *   `black-removebg-preview.png`: Black color variant[cite: 4].
    *   `blue-removebg-preview.png`: Blue color variant[cite: 4].
    *   `red-removebg-preview.png`: Red color variant[cite: 4].

---

## 💻 Tech Stack

*   **HTML5**: Semantic structure.
*   **CSS3**: Custom layouts and visual effects.
*   **JavaScript (ES6+)**: Dynamic DOM manipulation and interaction.

## ⚙️ Setup & Usage

To view the project locally:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/BMW-Website.git
    ```
2.  **Enter the project folder**:
    ```bash
    cd BMW-Website-main
    ```
3.  **Open the application**:
    Launch `index.html` in any modern web browser to explore the interactive showroom[cite: 4].
```
